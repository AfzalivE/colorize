## Usage

**Step 1:** Add the JitPack repository to your build file


```groovy
repositories {
    maven {
        url "https://jitpack.io"
    }
}
```


**Step 2:** Add the dependency in the form
```groovy
dependencies {
    compile 'com.github.cesarferreira:colorize:0.1.0'
}
```
